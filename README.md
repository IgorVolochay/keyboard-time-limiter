## Орисание программы
Программа ограничивает глобальный ввод текста с клавиатуры по времени. Программа блакирует ввод большего количества символов в заданный интервал времени. В обозначенное время можно вписать лишь 1 символ, далее необходимо ждать. Базово в программе указан интервал в `1 минуту`

## Как использовать
Для того чтобы использовать программу, необходимо установить модуль
- keyboard==0.13.5

а также запустить программу под Администратором в Windows системах и под Sudo в Linux системах
